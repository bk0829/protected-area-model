
# 종-면적 관계식 기반 보호구역 자동 추천 시스템

## 실행 방법

1. Python을 설치합니다.
2. 이 폴더에서 터미널을 엽니다.
3. 필요한 라이브러리를 설치합니다.

```bash
pip install -r requirements.txt
```

4. 앱을 실행합니다.

```bash
streamlit run app.py
```

## 사용 순서

1. 항공사진 또는 드론 사진을 업로드합니다.
2. 격자 크기를 정합니다.
3. 자동 생성된 지형행렬을 확인하고 필요하면 직접 수정합니다.
   - 0: 개발지역·도로
   - 1: 산림
   - 2: 습지·하천
   - 3: 초지·농경지
4. 각 지형의 c, k 값을 설정합니다.
5. 총 보호 셀 수를 정하고 최적화 버튼을 누릅니다.
6. 사진 위 추천 보호구역, 선택행렬, 예상 종수를 확인합니다.

## 모델

목적함수:

S = c_F A_F^(k_F) + c_W A_W^(k_W) + c_G A_G^(k_G)

제약조건:

A_F + A_W + A_G = A_total

각 A는 해당 지형에 실제로 존재하는 셀 수를 넘을 수 없습니다.

## 주의

이 앱은 수학 심화탐구용 단순화 모델입니다.
실제 보호구역 지정에는 현장 생태조사, 종 분포 자료, 토지 비용,
서식지 연결성, 희귀종 가중치 등이 추가되어야 합니다.
